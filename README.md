# ZobsAI Autofill Chrome Extension

## Overview
ZobsAI Autofill is a lightweight Chrome Extension built using
**JavaScript (ES6+)** and **Chrome Extension Manifest V3**.

The extension helps users automatically fill job application forms
with a single click, saving time and reducing repetitive manual typing.

This project demonstrates JavaScript fundamentals, DOM manipulation,
and Chrome Extension basics by solving a real-world problem.

---

## Why This Extension?
At ZobsAI, small tools that solve real problems matter.
This extension showcases:
- JavaScript fundamentals
- DOM selection & manipulation
- Chrome Extension (Manifest V3) architecture
- Practical problem-solving approach

---

## Features
- One-click form autofill
- Popup UI with **Autofill Form** button
- Automatically detects common input fields
- Works on most job application forms
- Clean, simple, and readable code
- Uses content scripts for DOM interaction

---

## Autofilled Fields
- Full Name
- Email Address
- Phone Number
- Skills / Experience

---

## Tech Stack
- JavaScript (ES6+)
- Chrome Extensions (Manifest V3)
- HTML & CSS

---

## Installation & Setup
1. Download or clone the repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the project folder
